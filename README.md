CRUD de Tareas

Pequeña aplicación hecha en JavaScript para practicar lo básico de un CRUD: agregar, editar y eliminar tareas. Es simple, funciona en el navegador y no usa librerías externas.

 Funcionalidades

Agregar tareas escribiendo en el input.

Editar tareas usando un botón que permite cambiar el texto.

Eliminar tareas con un clic.

Contador de tareas que se muestra por consola.

Todo se renderiza en pantalla sin recargar la página.

 Estructura del proyecto
/
├── index.html
├── style.css
└── app.js

 Cómo usarlo

Abre index.html en tu navegador.

Escribe una tarea y dale al botón Agregar.

Puedes editar o borrar cuando quieras.

En la consola del navegador verás el total de tareas.
